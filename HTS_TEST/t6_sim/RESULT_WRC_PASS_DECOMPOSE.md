## 시나리오별 증감 표

### 행 단위 (53조건)

동일 `(시나리오, 조건)` 행마다 **M0 Pass = M2 Pass**, **Δ = 0**. (CRC+·BER·판정 문자열도 동일.)

### 상위 시나리오 S1~S10 (Pass 합산, 동일 조건 집계)

| 시나리오 | M0 Pass 합 | M2 Pass 합 | Δ | 비고 |
|----------|------------|------------|---|------|
| S1 | 20 | 20 | 0 | 클린 1행 |
| S2 | 160 | 160 | 0 | 위상 8행×20 |
| S3 | 81 | 81 | 0 | AWGN 다단 (-30~10dB) |
| S4 | 140 | 140 | 0 | CW offset sweep 7행 |
| S5 | 140 | 140 | 0 | CFO 8행 (5000Hz 실패 1행 포함) |
| S6 | 80 | 80 | 0 | 페이딩 4행 |
| S7 | 66 | 66 | 0 | barrage 다단 (재밍) |
| S8 | 60 | 60 | 0 | 표준 재밍 JSR/P |
| S9 | 0 | 0 | 0 | Full 조합 |
| S10 (a+b+c) | 10200 | 10200 | 0 | Endurance + LPI |

**총합 Pass:** M0 = M2 = **10927**, **총합 Δ = 0**.  
(종합 표의 `정량 합계: 10927 / 11200`과 일치.)

## 가설 판정

- **패턴:** 지시문 **패턴 1**에 해당 — **모든 시나리오·모든 행에서 Δ = 0**. (패턴 2·3·4·5는 해당 없음: 재밍 전용 이득·CW만 이득·전면 악화·혼합 모두 아님.)

- **가설 A (Polar 등 하류가 chip 변형을 완전히 흡수하여 pass 불변):** 본 로그만으로 Polar 내부를 직접 증명하진 못하나, **“WRC로 인한 pass 변화가 시나리오 어디에도 나타나지 않는다”** 는 관측과 **합치**는 결론.

- **가설 D (재밍 시나리오만 유효):** **반증** — S7·S8 등 재밍 행에서도 **Δ = 0** (숨은 ± 상쇄로 재밍만 이득이 나는 패턴은 **관측되지 않음**).

- **WRC Method 2:** `total_delta`는 크나(별도 `wrc_ir_diag_2.log`), **이번 pass 분해에서는 효과 0** — 상쇄가 아니라 **전 구간 무변**.

## 자진 고지

- **로그:** `wrc_ir_method_0.log`, `wrc_ir_method_2.log` **존재·형식 일치** 확인. 재실행 불필요.
- **해석:** S10은 표에서 `S10a`/`S10b`/`S10c`로 나뉘어 있어, 상위 표에서는 **세 행의 Pass 합(10200)** 으로 묶음.
- **한계:** 본 분해는 **Pass(비트일치 trial 수)** 만 비교. 지연·CPU(총 소요) 차이는 Method 2가 큼 — pass 판정과 별개.
